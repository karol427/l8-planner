# L8 Planner

Operational Event Planner based on:
- React
- TypeScript
- Vite
- FullCalendar Scheduler

## Install

npm install

## Run

npm run dev

## Build

npm run build

## Deploy to GitHub

1. Create repo
2. Upload files
3. Enable Vercel / Netlify
